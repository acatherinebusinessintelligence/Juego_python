Juego Interactivo para Aprender Python

Reto 1a: Crea una variable llamada nombre y asígnale tu nombre como texto. Ejemplo: nombre = "Ana"

Reto 1b: Crea una variable llamada edad y asígnale tu edad como número entero. Ejemplo: edad = 20

Reto 1c: Crea una variable llamada altura con tu altura en metros (número decimal). Ejemplo: altura = 1.65

Reto 1d: Crea una variable booleana llamada es_estudiante. Ejemplo: es_estudiante = True

Reto 2: Crea una variable llamada ciudad y luego imprímela usando print().

Reto 3: Crea dos variables numéricas (a y b) y muestra su suma con print(a + b).

Reto 4: Usa una f-string para imprimir tu nombre con print(f"Hola, mi nombre es {nombre}").

Reto 5: Crea dos variables decimales (x y y) y muestra su multiplicación con print(x * y).

Reto 6: Imprime el resultado de una resta decimal usando format(). Ejemplo: precio = 100.0, descuento = 20.0, print("El precio final es: {:.2f}".format(precio - descuento))

Reto 7: Crea una lista llamada frutas con al menos tres elementos. Luego imprime el segundo elemento.

Reto 8: Crea un diccionario llamado persona con claves nombre, edad y ciudad. Luego imprime el valor de la clave nombre.
